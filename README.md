## 业务范围
- 国内贸易代理

- 互联网销售

## 网站结构
1. **首页 (index.html)** - Hero区域、服务概览、关于我们、新闻预览、联系CTA
2. **服务页面 (services.html)** - 详细业务服务和移动应用集成
3. **企业文化 (culture.html)** - 公司价值观、团队、使命
4. **新闻页面 (news.html)** - 公司动态、行业资讯
5. **联系页面 (contact.html)** - 联系表单、公司信息、地图
6. **隐私政策 (privacy.html)** - 全面的隐私政策，包含广告合规
7. **服务条款 (terms.html)** - 完整的服务条款
